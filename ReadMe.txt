1. для корректной работы нужна весрия Php 7.1;
2. проект использует БД, по-этом желательно использовать миграции :)

краткое описание:
- стартовая страница имеет форму для создания твитер-пользователя, твиты которого нужно выводить;
- также страница имеет таблицу, в которой отображаются уже созданные пользователи;
- для того чтобы посмотреть твиты пользователя нужно нажать текстовую ссылку "tweets";
- для того чтобы удалить пользователя из списка, нужно нажать текстовую ссылку "delete".
- если у пользователя нету твитов, система покажет сообщение, и вернет Вас на стартовую страницу.
